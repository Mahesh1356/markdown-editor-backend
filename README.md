# React Markdown editor 

Clone the repo using ssh
``` cmd
git@github.com:Mahesh1356/markdown-editor-backend.git
```

**Commands to run this app** run
```bash 
npm install
```

```bash 
npm run start:dev
```

To run test 

```bash 
npm run test
```

** *

**Note:** *Make sure that you are running frontend app*
